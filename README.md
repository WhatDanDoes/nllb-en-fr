# Setup

## Virtual Environments

From https://packaging.python.org/en/latest/guides/installing-using-pip-and-virtual-environments/

Create:

```
venv env
```

Activate:

```
source env/bin/activate
```

Deactivate:

```
deactivate
```

## Install dependencies

```
pip install -r requirements.txt
```
